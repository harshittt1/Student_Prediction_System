# 🎓 Student Intelligence System

A machine learning-based web application that predicts:

- 📊 Student Performance (Pass/Fail)
- 💼 Placement Status (Placed / Not Placed)

Built using Python, Flask, and Scikit-learn.

---

## 🚀 Features

- Two independent ML models:
  - Performance Prediction Model
  - Placement Prediction Model
- Real-time predictions through a web interface
- Clean UI with separate sections for performance and placement
- Basic suggestion system based on results

---

## 🧠 How It Works

### 📊 Performance Prediction
Takes inputs like:
- Study Hours
- Attendance
- Sleep Hours
- Previous Score
- Internet Access

Predicts whether a student will **Pass or Fail**.

---

### 💼 Placement Prediction
Takes inputs like:
- CGPA
- Internship Experience
- Communication Skills
- Projects Completed
- Extra Curricular Score

Predicts whether a student will be **Placed or Not Placed**.
